Decouple your package from implementation

Httplug allows you to write reusable libraries and applications that need an HTTP client without binding to a specific implementation. Read more about the why and how of the HTTP client abstraction in our documentation.

Diagram here

Use your HTTP Client of choice

We provide adapters for existing HTTP Clients, so you can use whatever client you want.

Separate logic using plugins

Need to log each request? Authenticate a request? It's easy. Our powerful plugin system allows you to add custom logic to the request's lifecycle.
